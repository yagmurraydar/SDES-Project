# S-DES Cryptography Project

This project implements the Simplified Data Encryption Standard (S-DES).

## Features
- Key generation (K1, K2)
- Encryption / Decryption
- ECB, CBC, OFB modes
- Brute-force attack simulation

## Usage
1. Run the program
2. Enter key and plaintext
3. Select mode (ECB, CBC, OFB)
4. Click Encrypt / Decrypt
